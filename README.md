## CustomHashMap

Пользовательская реализация хеш-таблицы (HashMap) на Java, демонстрирующая принципы работы хеш-коллекций.

Методы: 

`get(K key)` - Возвращает значение по ключу.

`put(K key, V value)` - Добавляет пару ключ-значение в карту. Возвращает предыдущее значение, связанное с ключом, или null, если ключа не было

`remove(K key)` - Удаляет элемент по ключу. Возвращает удалённое значение или null, если ключ не найден

`size()` - возвращает колличество пар (int) хранящихся в CustomHashMap
